Conclusión

Al llegar a la finalización de este proyecto hemos comprobado que el proceso para desarrollar un sistema para el control escolar es complejo y que se tienen que tomar en cuenta varios factores y este es un trabajo de equipo, no solo de uno o dos integrantes, tienen que cooperar todos y el trabajo va a ser más fácil al desarrollar las peticiones de nuestro cliente. Con este proyecto se busca cubrir las necesidades del cliente, en este caso de la institución, y si es posible darle un plus a su software para que nuestro cliente tenga facilidad en todos los sentidos, este sistema comprende varios componentes para poder ser llevados a cabo apartir del análisis el cual nos ayuda a traducir las necesidades de nuestro cliente, en una empresa, el análisis y diseño de sistemas es más que nada estudiar la sutuación o necesidad con la finalidad de hacer el mejor trabajo para la satisfacción de nuestro cliente
